CT Commerce Lite social sharing

== Description ==

Social sharing for CT Commerce Lite ecommerce products



== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/ctcl-sharing` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Where is this block best suited to be used?=

It is for providing sharing option for different social media sites for CT Commerce Lite product page
